# Example Apps

This directory groups the language-specific example apps for Waymark.

- [python](</Users/piercefreeman/projects/waymark/example-app/python/README.md>) is the current end-to-end example. It includes a FastAPI webapp, Docker setup, and tests that exercise the existing Python workflow/action runtime.
- [javascript](</Users/piercefreeman/projects/waymark/example-app/javascript/README.md>) is the first Next.js example. It demonstrates the intended JavaScript authoring model and compiler wiring for workflows and actions.
